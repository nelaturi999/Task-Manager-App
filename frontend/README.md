# Task Manager App

## Project Description

Task Manager App is a full-stack web application where users can register, login, and manage their tasks across three stages: Todo, In Progress, and Done. Users can create, update, and delete tasks. The project includes authentication, JWT-based protected APIs, MongoDB database integration, and a clean responsive user interface.

## Features

- User registration
- User login
- JWT authentication
- Protected dashboard
- Create tasks
- Update task stage
- Delete tasks
- Todo, In Progress, Done stages
- MongoDB database storage
- Responsive design
- Loading and error states

## Tech Stack
